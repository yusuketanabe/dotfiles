iceberg-syntax
==============


Overview
--------
Iceberg is a dark blue syntax theme for Atom, originally for Vim.
It brings a cool-headed perspective for your happy coding :)

See the [official site](http://cocopon.github.io/iceberg.vim/) for more
information.
